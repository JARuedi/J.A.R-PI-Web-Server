# J.A.R-PI-Web-Server

Utilizing a Raspberry PI five computer chip, a Penta Sata Hat, allowed the chip to hold half a terabyte of data, using NGINX, Node.js, Maria Databases, this was able to containerize and host BSU’s CS Students Software Projects and contained each one in their own Docker Container.

Using Python Scripting to send and automate Data from MariaDB to OfficeLibre for easy readability for developers.
